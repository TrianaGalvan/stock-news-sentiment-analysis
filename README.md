# stock-news-sentiment-analysis
Android app that applies sentiment analysis to tweets and news tittles of stocks. 

## Technologies 

The app was built with the Firebase Android SDK. 

Firebase services used: 
* Authentication
* Storage
* Firestore
* ML Kit
